# Промпт для наполнения раздела "Новости"

Используй этот промпт с Claude или другим LLM чтобы собрать реальные новости для проекта.

---

## Промпт для поиска закон-апдейтов

```
Найди актуальные правовые обновления и новости Беларуси за 2024-2026 год.

Источники для поиска:
- pravo.by — Национальный правовой интернет-портал
- nalog.gov.by — Министерство по налогам и сборам
- minfin.gov.by — Министерство финансов
- mintrud.gov.by — Министерство труда
- mzsr.gov.by — Министерство здравоохранения
- portal.gov.by — Единый государственный портал
- mvd.gov.by — МВД (для реестра экстремистских материалов)
- ssf.gov.by — ФСЗН

Для каждой новости/закона выдай JSON следующей структуры:
{
  "id": "news_<порядковый_номер>",
  "title": "Название закона или новости",
  "category": "<taxes|home|docs|family|work|law_update|extremist_registry>",
  "category_name": "<человекочитаемое название категории>",
  "date": "<DD.MM.YYYY>",
  "target": "<кого касается>",
  "short": "<краткое описание, 1-2 предложения>",
  "what_to_do": "<что нужно сделать гражданину>",
  "source_url": "<прямая ссылка на источник>",
  "status": "published",
  "priority": "<high|medium|low>",
  "related_scenarios": [],
  "related_problems": [],
  "profile_tags": ["<документы|налоги|семья|работа|жкх|авто|ип>"],
  "processing_status": "reviewed"
}

Собери минимум 15 записей, распределив по категориям:
- 5 записей: law_update (изменения в законодательстве с номерами законов)
- 3 записи: taxes (налоговые изменения)
- 3 записи: family (семья, дети, пособия)
- 2 записи: docs (документы и регистрация)
- 2 записи: extremist_registry (актуальные обновления реестра МВД)

Важно:
- Только реальные события с реальными датами
- Ссылки только на официальные государственные сайты
- Если точная ссылка неизвестна — давай ссылку на главную страницу ведомства
- Пиши по-русски, простым языком для обычных граждан

Выведи результат как валидный JSON массив.
```

---

## Как использовать результат

1. Скопируй JSON-массив из ответа LLM
2. Открой `src/data/mock_data.py`
3. Добавь записи в конец массива `LEGAL_UPDATES = [...]`
4. Проверь что у каждой записи есть все обязательные поля
5. Перезапусти приложение (`scripts/run_web.sh`)

---

## Категории и их коды

| Код | Название | Описание |
|-----|----------|----------|
| `law_update` | Закон-апдейт | Изменения в законах с номерами актов |
| `extremist_registry` | Реестр | Обновления реестра МВД |
| `taxes` | Налоги | Налоговые изменения |
| `home` | ЖКХ | Жильё и коммунальные услуги |
| `docs` | Документы | Паспорта, справки, регистрация |
| `family` | Семья | Дети, пособия, брак |
| `work` | Работа | Трудовые отношения |

## Profile tags (для таргетинга)

Используй эти теги в поле `profile_tags` для таргетированных уведомлений:

- `documents` — кому нужно переоформить документы
- `taxes` — ИП и плательщики налогов
- `family` — семьи с детьми
- `housing` — собственники жилья
- `housing_owner` — собственники (более точный)
- `auto` — владельцы авто
- `ip` — индивидуальные предприниматели
- `student` — студенты
- `retired` — пенсионеры
- `utility` — плательщики ЖКХ
