# IDEA — Белпомощник

## Суть продукта

Мобильное приложение для граждан Республики Беларусь. Пользователь сталкивается с жизненной ситуацией (потеря паспорта, рождение ребёнка, открытие ИП, переезд и т.д.) и получает понятный пошаговый план: что делать, какие документы нужны, какие сроки, куда обращаться, каких ошибок избегать.

## Целевая аудитория

Обычные граждане Беларуси, которые не знают, как действовать в бюрократических ситуациях.

## Ключевая ценность

- Не нужно самому искать законы и инструкции — всё уже структурировано
- Персональный план прогресса по ситуации
- Хранение личных документов с напоминаниями о сроках
- Актуальные изменения законодательства, которые касаются конкретного пользователя

## Основные сценарии пользователя

1. Выбрать жизненный сценарий из каталога → создать личную ситуацию → отмечать задачи по шагам
2. Найти проблему в каталоге → прочитать инструкцию → сохранить себе
3. Добавить личный документ → получать напоминания до истечения срока
4. Просмотреть изменения законодательства → узнать, что нужно сделать
5. Администратор/редактор → создавать и публиковать сценарии, закон-апдейты
