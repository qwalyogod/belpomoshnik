# Обучающий модуль

## Статус

Обучающий модуль реализован как вспомогательная MVP-функция на моковых данных и локальном состоянии приложения.

Он не является основной функцией проекта. Основная функция «Белпомощника» - справочник жизненных ситуаций и понятные пошаговые планы.

В первой бете модуль используется как дополнительная демонстрационная фишка: руководителю можно показать переключатель режима обучения в профиле, экран `/learning`, достижения и микро-тест по теме «Потерял паспорт».

## Режим обучения

- В `ProfilePage` есть переключатель «Режим обучения».
- Если режим выключен, карточка проблемы работает как обычный справочник.
- Если режим включен, в конце `ProblemDetailPage` показывается блок «Проверьте себя».
- Состояние переключателя сохраняется локально в `data/app_state.json`.
- После подключения полноценного backend настройку планируется сохранять в профиле пользователя на сервере.

## Микро-тест

Сейчас добавлен демо-тест для темы «Потерял паспорт».

Тест содержит 3 вопроса:

- выбор первого правильного действия при утере паспорта;
- выбор документов, которые могут понадобиться для восстановления;
- выбор правильной последовательности действий.

Drag-and-drop пока не используется. Для последовательности пользователь выбирает один из готовых вариантов.

## Результат теста

После проверки ответов показывается:

- результат в формате «Ваш результат: 2 из 3»;
- уровень подготовки:
  - `0-1`: «Стоит повторить тему»;
  - `2`: «Базовый уровень»;
  - `3`: «Вы хорошо подготовлены»;
- кнопка «Пройти ещё раз».

Результат теста пока хранится только в текущей сессии. Сохранение результата в профиль планируется после появления backend-API для обучения.

## LearningProgressPage

Добавлен экран `LearningProgressPage` по маршруту `/learning`.

Экран содержит:

- заголовок «Обучение»;
- пояснение, что это дополнительный режим;
- статистику: прочитано материалов, пройдено тестов, средний результат;
- прогресс по категориям: Документы, ЖКХ, Налоги, Семья, Работа, Здоровье;
- достижения.

## Достижения

В моковых данных есть достижения:

- Первый шаг;
- Документы под контролем;
- Разбираюсь в ЖКХ;
- Закон в курсе;
- Готов к переезду;
- Без паники.

В `ProfilePage` добавлен блок «Достижения» и переход «Подробнее об обучении» на `/learning`.

## Ограничения

- Нет backend-API для результатов обучения.
- Нет серверного хранения прогресса обучения.
- Нет реального сохранения результатов тестов.
- Нет полноценной backend-авторизации.
- Все данные берутся из `src/data/mock_data.py`.
