# Чеклист приёмки первой беты

Дата: 2026-05-19.

## Запуск

- [ ] Приложение запускается командой `.venv/bin/flet run src/main.py`.
- [ ] Нет traceback при старте.
- [ ] Web-preview запускается командой `.venv/bin/flet run --web src/main.py`.
- [ ] QR/iPhone-запуск доступен командой `.venv/bin/flet run --ios src/main.py`.

## Основная навигация

- [ ] Главная открывается.
- [ ] Нижняя навигация работает: Главная, Сценарии, Ситуации, Уведомления, Профиль.
- [ ] Маршруты `/scenarios`, `/scenario-detail`, `/situations`, `/situation-detail`, `/documents`, `/notifications`, `/profile`, `/learning`, `/admin` не падают.

## Пользовательский путь

- [ ] В каталоге есть минимум 3 жизненных сценария.
- [ ] Сценарий «Рождение ребёнка» заполнен этапами, задачами, документами, организациями и источниками.
- [ ] Кнопка «Создать мою ситуацию» создаёт ситуацию из шаблона.
- [ ] Задачи добавляются в созданную ситуацию.
- [ ] Документы отображаются внутри задач и в блоке подготовки документов.
- [ ] Прогресс считается автоматически по выполненным задачам.
- [ ] Данные сохраняются после перезапуска.

## Основные экраны

- [ ] Главная выглядит как дашборд.
- [ ] «Мои ситуации» показывают статусы, прогресс, фильтры и количество задач.
- [ ] Детальная ситуация позволяет отмечать, редактировать и удалять задачи.
- [ ] Документы отображаются, редактируются и удаляются.
- [ ] Уведомления отображаются и отмечаются прочитанными.
- [ ] Профиль отображает настройки, достижения и демо-сброс.
- [ ] Обучение доступно по `/learning`.
- [ ] Админ-панель отображает концепцию управления контентом.

## iPhone layout

- [ ] Контент не залезает под status bar, notch и Dynamic Island.
- [ ] Нижняя навигация не конфликтует с home indicator.
- [ ] Последние элементы длинных экранов не прячутся под нижнюю навигацию.
- [ ] Диалоги помещаются на ширине 375 px.
- [ ] Нет случайного горизонтального скролла.

## Ограничения беты

- [ ] Пользователь понимает, что данные пока локальные.
- [ ] Юридические формулировки помечены как демонстрационные и требуют финальной проверки.
- [ ] Backend/API для мобильного клиента подключён только частично.
