# Дизайн-система

## Общий стиль

- Современный.
- Светлый.
- Минималистичный.
- Понятный для пользователя без технического опыта.
- Основной интерфейс - карточный.

## Цвета

| Назначение | Цвет |
| --- | --- |
| background | `#F6F8FB` |
| surface | `#FFFFFF` |
| primary blue | `#2563EB` |
| primary green | `#16A34A` |
| text dark | `#111827` |
| text muted | `#6B7280` |
| border | `#E5E7EB` |
| warning | `#F59E0B` |
| danger | `#EF4444` |

## Скругления

- Карточки: `16px`.
- Кнопки: `12px`.
- Поля ввода: `12px`.

## Отступы

- Page padding: `16px`.
- Card padding: `16px`.
- Gap: `12px` или `16px`.

## Типографика

- Заголовки крупные и четкие.
- Основной текст читаемый.
- Минимум мелкого текста.
- Русский язык интерфейса обязателен.

## Компоненты

- Нижняя навигация.
- Карточный интерфейс.
- Крупные кнопки.
- Поля ввода с понятными подсказками.
- Переиспользуемые карточки проблем, документов, уведомлений и ситуаций.

## Доступность

- Крупный шрифт.
- Высокий контраст.
- Понятные подписи.
- Нажимаемые элементы должны быть достаточно крупными.
- Важные действия должны иметь ясный текст.
