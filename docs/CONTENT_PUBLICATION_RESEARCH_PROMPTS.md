# Промпты для массового наполнения новостей, закон-апдейтов и экстремистских материалов

Дата: 2026-06-13.

Назначение документа: подготовить три больших промпта для ChatGPT/Deep Research, чтобы получить много проверяемого официального контента для разделов «Новости», «Закон-апдейты» и «Экстремистские материалы» за период `2022-01-01` — `2026-06-08`.

Цель: ChatGPT должен самостоятельно искать, сверять и структурировать материалы так, чтобы Codex мог затем выполнить структурную проверку JSON и импортировать данные в проект без ручного переписывания каждой записи.

## Как запускать промпты

В ChatGPT перед отправкой промпта включить:

- Deep Research, если доступен;
- поиск в интернете;
- цитирование или отображение найденных источников;
- запрет на использование только памяти модели;
- проверку каждой ссылки перед добавлением записи в JSON.

Если ChatGPT не может открыть официальный источник или подтвердить дату, запись нужно отправлять в `rejected_items`, а не додумывать.

## Куда сохранять результат

Результат каждого промпта сохранить в `data/import/`:

- новости: `data/import/content_news_calendar_2022_2026.json`;
- закон-апдейты: `data/import/content_law_updates_2022_2026.json`;
- экстремистские материалы: `data/import/content_extremist_materials_2022_2026.json`.

После получения файлов проверить:

```bash
.venv/bin/python scripts/validate_content_batch.py data/import/content_news_calendar_2022_2026.json
.venv/bin/python scripts/validate_content_batch.py data/import/content_law_updates_2022_2026.json
.venv/bin/python scripts/validate_content_batch.py data/import/content_extremist_materials_2022_2026.json
```

Затем импорт:

```bash
.venv/bin/python scripts/import_content_batches.py data/import/content_news_calendar_2022_2026.json data/import/content_law_updates_2022_2026.json data/import/content_extremist_materials_2022_2026.json
```

## Общий контракт JSON

Каждый ответ должен быть валидным JSON-объектом без markdown-обёртки:

```json
{
  "batch_meta": {
    "project": "Белпомощник",
    "generated_for": "content_import",
    "category_id": "news_calendar",
    "period_from": "2022-01-01",
    "period_to": "2026-06-08",
    "checked_at": "2026-06-08",
    "requires_human_review": true
  },
  "official_sources_used": [],
  "problems": [],
  "scenarios": [],
  "news_articles": [],
  "law_updates": [],
  "extremist_entries": [],
  "source_candidates": [],
  "rejected_items": []
}
```

Общие правила:

- возвращать только JSON;
- не добавлять комментарии вне JSON;
- не использовать даты позже `2026-06-08`;
- не выдумывать URL;
- не брать материалы из частных блогов, форумов, Telegram-каналов, соцсетей и перепечаток;
- `cover`, `gallery`, `cover_url`, `media_urls` заполнять только прямыми `https://` ссылками на официальные изображения/медиа, если они найдены на официальном источнике;
- если прямой официальный image URL не найден, оставить поле пустым, не использовать стоки;
- каждая статья должна иметь краткое описание для карточки и полноценный текст для detail-страницы;
- новости должны иметь `views_seed` от `120` до `18000`, чтобы в админке и редакторской панели были заметны просмотры;
- новости импортируются автором `Тестовый редактор` через `scripts/import_content_batches.py`;
- все создаваемые материалы должны иметь опубликованный статус, если они подтверждены официальным источником.
